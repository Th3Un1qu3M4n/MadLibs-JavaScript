Problem # 01

Problem Description – Personal Narrative

For this assignment, you will create a webpage about yourself. The web page must fulfill the following requirements:

  1. Create simple HTML forms with input elements and a button,
  2. Write Javascript functions that are used as event handlers,
  3. Write JQuery statements that gather the information from the input fields, store the values into variables, and perform some operations with them,

Task 1: Creating the HTML page

Create the file assign2.html
  1. Create the Home page where the assignment question is displayed.
  2. Create another page name “Story Creation” that takes a form as input from user. Form input contains person name, character name, adverb and adjective as text fields. The form contains a button “ Tell Story”.
  3. it'll be useful to show a default value in all input elements to simplify your testing of the page.
  4. The form should also have a < button > element with an ID so that you can attach the event handler to it. It should also have the attribute type="button".
  5. There should also be a < div > element with an ID where the story will appear.
  6. The requirements for the story and its appearance are:
    1. There must be exactly four input fields, but their meaning is up to you. Give them default values so that the form is easier to test.
    2. Define a new CSS rule in assign2.css to style all the user-supplied words, so that you can see which words were chosen when the story is displayed. You may use whatever color, font, etc. you wish - just make sure the inserted words are distinguishable from the rest of the story.
    3. The story must be at least four, but not more than six, sentences long (no longer!).
    4. There should be 8 missing words in the story, so you will have to use some of the user-supplied words more than once. For example, the person's name was used three times in the sample story (Note that duplicated words are entered only once by the user!)
    5. Define HTML elements where your JavaScript code will display the values from the form. Think about how you'll target the correct element for the matching value.
    6. A story title should appear at the top of the story. This story title should include your own name (if you are Aslam, something like “Aslam's Crazy Story!”). Use your story title as the < title > for your HTML page as well.
The content of the story can be anything that you like — be creative!

Event Handler:

In your assign2.js file, define a function (name it whatever you like) that will process the form. It should
  • Read the four form inputs using the jQuery .val() method, storing the values in variables.
  • Copy the values into the story using the jQuery .html() method. In your Javascript file, add code to attach the function to the button that will be clicked.
